# tasseff.com

[![Build Status](https://travis-ci.org/tasseff/tasseff.com.svg?branch=master)](https://travis-ci.org/tasseff/tasseff.com)

Clone it

```sh
git clone https://github.com/tasseff/tasseff.com.git
```

Install Jekyll

```sh
gem install jekyll
```

Run Jekyll within the repo

```sh
cd tasseff.com
jekyll serve --watch
```
